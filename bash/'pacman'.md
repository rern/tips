**Update**  
```sh
pacman -Sy
```

**Update then install**  
```sh
pacman -Sy package
```

**Install**  
```sh
pacman -S package
```

**Silent install**  
```sh
pacman -S --noconfirm package
```

**Download only**  
```sh
pacman -Sw package
```

**Install from file**  
```sh
pacman -U package_file
```

**System wide upgrade**  
```sh
pacman -Syu
```
