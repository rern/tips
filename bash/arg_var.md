argument / variable
---

`$var` is simplified `${var}`    

`$@` : all arguments    
`$1` : Nth argument    
`$?` : last exit code    

`$@` `$*` : all arguments as separated words  
`"$@"` : all arguments as each words  
`"$*"` : all arguments as a single word  

`"${array[@]}"` : all value as separated words  
`"${array[*]}"` : all arguments as a single word
